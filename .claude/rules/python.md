# Python Rules

Applies to all `.py` files generated by any agent.

## Style (PEP 8 + Black)
- Formatter: `black` (line length 88)
- Linter: `ruff` (replaces flake8 + isort + pyupgrade)
- Type checker: `mypy --strict`
- All three must pass with zero errors before code is considered complete

```toml
# pyproject.toml
[tool.black]
line-length = 88

[tool.ruff]
line-length = 88
select = ["E", "F", "I", "UP", "B", "SIM"]

[tool.mypy]
strict = true
```

## Type Annotations
```python
# All functions must have full type annotations
from typing import Optional

def get_user(user_id: str) -> Optional[User]:
    ...

# Use | for unions (Python 3.10+)
def process(value: int | str) -> None:
    ...

# Prefer dataclasses or Pydantic over plain dicts for structured data
from dataclasses import dataclass

@dataclass
class UserInput:
    email: str
    password: str
    name: str
```

## Error Handling
```python
# Custom exceptions — always inherit from a base AppError
class AppError(Exception):
    def __init__(self, message: str, code: str) -> None:
        super().__init__(message)
        self.code = code

class NotFoundError(AppError):
    def __init__(self, resource: str) -> None:
        super().__init__(f"{resource} not found", "NOT_FOUND")

# Never catch bare Exception without re-raising or logging
try:
    result = await fetch_user(user_id)
except NotFoundError:
    raise
except Exception as e:
    logger.exception("Unexpected error fetching user", user_id=user_id)
    raise
```

## Async
```python
import asyncio

# Use async/await throughout — never mix sync and async I/O
async def create_user(input: UserInput) -> User:
    existing = await user_repo.find_by_email(input.email)
    if existing:
        raise ConflictError("Email already registered")
    return await user_repo.create(input)

# Run concurrent tasks with gather
results = await asyncio.gather(fetch_a(), fetch_b(), fetch_c())
```

## Dependencies
```python
# Use Pydantic v2 for validation at API boundaries
from pydantic import BaseModel, EmailStr, field_validator

class CreateUserRequest(BaseModel):
    email: EmailStr
    password: str
    name: str

    @field_validator("password")
    @classmethod
    def validate_password(cls, v: str) -> str:
        if len(v) < 8:
            raise ValueError("Password must be at least 8 characters")
        return v
```

## Rules
- No mutable default arguments: `def fn(items: list = [])` → use `None` and create inside
- No star imports: `from module import *`
- No `print()` in production code — use `structlog` or `logging`
- f-strings over `.format()` and `%` formatting
- Use `pathlib.Path` over `os.path` for file operations
